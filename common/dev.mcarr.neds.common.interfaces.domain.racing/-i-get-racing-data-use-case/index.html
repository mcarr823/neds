IGetRacingDataUseCase

interface IGetRacingDataUseCase

Interface which should be implemented by any UseCase classes which retrieve data from an IRacingRepository implementation.

Data is queried from the repository, then stored in a flow.

The flow is then utilized by a viewmodel in order to expose the data to the user interface.
